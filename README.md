# 🚀 PHP & JS Dev-Env Blueprint

Ein High-End-Grundgerüst für moderne, sichere und hochperformante Web-Applikationen. Dieses Template kombiniert die striktesten PHP-Standards mit modernster Frontend-Tooling-Infrastruktur.

## 🌟 Highlights

- **PHP 8.4 Ready:** Automatische Modernisierung via Rector & statische Analyse via PHPStan (Level 6+).
- **Next-Gen Frontend:** Vite-Bundler, Biome (Linter/Formatter) und blitzschnelle Asset-Pipeline.
- **Quality First:** Architektur-Checks (Deptrac), Mutation-Testing (Infection) und A11Y-Audits (axe-core).
- **Developer Experience:** Farbige CLI-Tools, automatisierte Releases und vorkonfigurierte VS Code Umgebung.

## ⚡ Quick Start

```bash
# 1. Repository klonen & in den Ordner wechseln
git clone <url> && cd <folder>

# 2. Initiales Setup ausführen (erstellt .env, installiert Dependencies)
composer setup

# 3. Development Server starten
npm run dev
```
